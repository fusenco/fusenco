# DESIGN.md

## 品牌与视觉方向
- 品牌名：FUSEN
- 定位：面向海外游客的中国地陪服务独立站
- 风格：东方文化感与高端商务融合
- 气质：专业、可信赖、有文化底蕴、国际化

## Design Tokens

### 色彩
- 主色（深红）：#8B1A1A（中国朱红，品牌主色调）
- 主色浅：#A52A2A
- 金色：#C9A961（奢华点缀，用于线条/图标/强调）
- 金色浅：#D4BC7E
- 深色背景：#1A1410（暖色调炭黑，用于深色区域）
- 深色替代：#2C2420
- 米白背景：#F8F5F0（页面主背景）
- 纯白：#FFFFFF（卡片背景）
- 次要文字：#6B5D52
- 边框色：#E5DDD3

### 字体
- 标题字体：Cormorant Garamond（优雅衬线体，高端感）
- 正文字体：Inter（现代无衬线，清晰易读）
- 中文标题：Noto Serif SC

### 间距
- Section 垂直间距：py-20 ~ py-32
- 容器最大宽度：max-w-7xl
- 卡片内边距：p-6 ~ p-8

### 圆角
- 卡片：rounded-xl (0.75rem)
- 按钮：rounded-full
- 图片：rounded-lg

### 阴影
- 卡片悬浮：shadow-lg + 自定义暖色阴影
- 按钮无阴影，用边框和色彩区分

### 动效
- 滚动渐入：fade-in + slide-up
- 卡片悬浮：translate-y + shadow 加深
- 过渡时长：300ms ease

## 布局与响应式
- 桌面端：多列网格，宽屏体验
- 平板：2列网格
- 移动端：单列堆叠，汉堡菜单
- 断点：sm(640) / md(768) / lg(1024) / xl(1280)

## 多语言
- 支持 8 种语言：EN / RU / JA / KO / ES / PT / FR / AR
- 阿拉伯语需要 RTL 布局
- 语言切换器在导航栏右侧

## 组件规范
- 导航栏：透明→滚动后白底带阴影，Logo 左/链接中/语言+CTA右
- Hero：全屏背景图 + 暗色遮罩 + 居中文字 + CTA 按钮
- 服务卡片：白底 + 图标 + 标题 + 描述 + 悬浮金色边框
- 目的地卡片：图片 + 城市名 + 简述 + 悬浮放大
- 导游卡片：头像 + 姓名 + 语言标签 + 简介
- 联系区：WhatsApp/Email 按钮 + 简单询价表单
